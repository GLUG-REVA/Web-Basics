Day 2: Open source session

We will look at how to style your pages today.



As said yesterday, we use CSS to style our html documents. We saw how you could use the style tag for the same. But, today we shall se how you can use an external css file to apply styles to your page.

We use the link tag to import the css file. The link tag must be specified within the head tag.

Give below is an example

       
            <head>
                <link rel="stylesheet" href="master.css">
            </head>

The rel attribute stands for relation. It specifies the relation of the file with the html document. Here, it states the relation as 'stylesheet'. As simple as it looks. But remember the it must be specified within the head tag. Another point to note is that there is no closing tag for link

Alright, now that we now how to import the css file in your html page; we will look at how to write css. This begs the introduction of a new concept 'A CSS class'.


A css class is just a block of code that specifies that properties and their values. Let us look at an simple and familiar example.

All the code given below must exist in a .css file. In our case let us call it master.css 

        
            .back {
                background-color: red;
            }

Let us break it down now...


So, the first thing you need to observe is the CSS class defination.
'back' is the name of the class.
The next point is that the name of a CSS class always, always, starts with a dot(.)

The body of the class is enclosed within the curly braces({}).

The back class says that: change the background color to red.

Now you can how more than one property in the back class. Let us extend the class.

        
            .back {
                background-color: red;
                text-align: center;
                font-size: 20px;
            }

Phew, now we know how to declare a css class. Again, this was done in the msater.css file. Now, the next question is how do we use this class?

The answer to the question is pretty easy, Observe the following example.

        
            <h1 class="back">Hello, World</h1>

As you can see the usage we simply specify an attribute called class and give it the class name within quotes. The output is as follows:



Hello, World



You will notice that all the properties that we specifiedhave been applied.

Without the style applied to it the output would be as follows:

Hello,World

Let us define a new css class as follows:

        
            .ex1 {
                padding: 20px;
                width: 100px;
                height: 100px;
            }

We can apply more than one class to the same tag. The second one is given a preference over the first one. The way to do this is as follows:

        
            <h1 class="back ex1">Hello, World</h1>

The above statement is rendered as follows in the browser.

Hello, World

Now let us change padding to 50px and the width to 300px and the height to 400px. Theoutput shall be as follows:

Padding is the space between the text and the boundary of the object. The width and height are what they mean.

Hello, World

A CSS class can be used with any tag. Here is anothe example:

        
            <ol class="back">
                <li>One</li>
                <li>Two</li>
            </ol>

The output is as follows:

  1. One
  2. Two

That is some ugly output but, I hope it serves as an example.
